# Gravii Partner LP

Productized Gravii partner landing page built with Next.js App Router, React 19, and TypeScript.

## Commands

```bash
bun install
bun run dev
bun run build
bun run lint
bun run typecheck
```

## Structure

- `src/app`: route entrypoints, layout, metadata, and global styles
- `src/components/landing`: landing-specific interactive components
- `src/lib/landing-content.ts`: typed content model for all sections

## Notes

- The legacy single-file prototype remains at `Partner_Landing_new.html` for reference.
- The new app removes broken prototype links and replaces them with working docs or in-page actions.
