# Boundaries

This app owns the public Gravii landing page and the partner acquisition route.

## In Scope

- Landing page sections, layout, motion, and responsive behavior
- Waitlist server action and client form integration
- Partner route UI and handoff links
- Metadata routes and deployment-safe Next.js configuration
- App-local styles, assets, and docs

## Out of Scope Without Explicit Request

- Runtime changes in sibling apps
- Moving app code into shared packages
- Replacing the Landing API contract
- Framework/package-manager upgrades
- Reintroducing generic starter CMS, shop, or global canvas systems

## Default Rule

When UI and behavior conflict, preserve Gravii production behavior and make the UI match the approved landing design.
