# Domain Types

Shared frontend domain models that are stable enough to use across multiple Gravii apps.

Current scope:

- campaign concepts
- partner concepts
- population concepts
- risk concepts
- analytics concepts
- tier and label concepts

Non-goals for now:

- UI component props
- visual tokens
- strict API-generated response contracts
- app-shell or layout models

These types are intentionally normalized and lower-risk than the current app-local mock shapes.
